# myblog

## 依赖下载
```
npm install
```

### 项目运行
```
npm run serve
```

### 打包
```
npm run build
```


### 插件
采用了v-md-editor插件读取本地的md文件
```
npm i @kangc/v-md-editor -S
```
### 部署
采用vercel实现自动化部署
```
https://vercel.com/login?next=%2Fdashboard
```
